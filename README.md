# QA Film MVC Base

A starter Node.js project structured using the MVC pattern with Express and EJS.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the server:
   ```bash
   npm run dev   # uses nodemon
   npm start     # normal start
   ```
3. Open your browser at `http://localhost:3000`.

## Project Structure

```
/controllers  - controller logic
/models       - data models (currently empty)
/routes       - route definitions
/views        - EJS templates
/public       - static assets
app.js        - entry point
package.json  - project metadata
```

Feel free to expand the models, controllers, and routes as needed.
